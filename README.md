# Читалка (PDF Reader)

Простое Android-приложение для чтения PDF-файлов с локальным хранением книг.  
Поддерживает регистрацию/вход (локально, без сервера), загрузку PDF из устройства, просмотр страниц.

## Основные возможности

- Регистрация и вход (данные сохраняются локально в DataStore)
- Главный экран с фоном
- Экран «Мои книги» — список загруженных PDF
- Автоматическое копирование выбранных PDF во внутреннюю папку приложения
- Просмотр PDF с постраничной навигацией (предыдущая/следующая)
- Подсчёт количества страниц в каждой книге
- Bottom Navigation Bar (Домой / Книги)

## Требования

- Android Studio Iguana | 2023.2.1 или новее (или более свежая версия)
- Minimum SDK: API 24 (Android 7.0)
- Kotlin 1.9+ / 2.0
- Jetpack Compose

## Как запустить проект

### 1. Клонирование или скачивание

Скачай или клонируй репозиторий:

```bash
git clone https://github.com/wirix/book-sfu.git
# или просто скачай ZIP и распакуй
```

Открой папку проекта в Android Studio.

### 2. Синхронизация Gradle

После открытия проекта Android Studio предложит «Sync Project with Gradle Files» — нажми на слоника или выполни:

- File → Sync Project with Gradle Files

Дождись завершения (может занять 1–3 минуты при первом запуске).

### 3. Запуск на эмуляторе или устройстве

1. Подключи устройство по USB (режим разработчика + отладка по USB)  
   или  
   Создай/запусти эмулятор (рекомендуется API 30+)

2. Нажми зелёную кнопку **Run** (Shift + F10)  
   или выбери конфигурацию «app» и запусти

### 4. Добавление тестового PDF (для проверки)

**Вариант A — самый простой (drag & drop):**

- Скачай любой PDF на компьютер
- Перетащи файл мышкой прямо в окно запущенного эмулятора → попадёт в папку Downloads

**Вариант B — через Device File Explorer:**

- В Android Studio → View → Tool Windows → Device File Explorer
- Перейди в `/sdcard/Download/`
- Правой кнопкой → Upload → выбери PDF с компьютера

### 5. Как пользоваться приложением

1. При первом запуске — экран приветствия → нажми «Начать читать» → зарегистрируйся
2. После входа — попадёшь на экран «Мои книги»
3. Нажми «Загрузить PDF» → выбери файл из Downloads (или любой другой папки)
4. Кликни на добавленную книгу → откроется просмотр страниц
5. Листай страницы кнопками «Предыдущая» / «Следующая»

## Возможные проблемы и решения

| Проблема                                 | Решение                                                                 |
|----------------------------------------|-------------------------------------------------------------------------|
| «Unresolved reference: PdfRenderer»    | Убедись, что minSdk ≤ 21 (PdfRenderer с API 21)                        |
| Экран загрузки пустой                  | Проверь, что PDF-файл действительно скопирован в `files/pdfs/`         |
| Нет кнопки «Назад» в TopAppBar         | Добавь `@OptIn(ExperimentalMaterial3Api::class)` над composable        |
| Долгая загрузка больших страниц        | Это нормально для PdfRenderer без кэширования. Для оптимизации нужна библиотека |
| Файл не отображается после выбора      | Перезапусти приложение (DataStore иногда обновляется не мгновенно)     |
